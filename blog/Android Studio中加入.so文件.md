# Android Studio加入.so文件 #

---------------------------------------

Android Studio中加入.so文件与Eclipse中有所不同。在Android Studio中加入.so文件的位置为：   


>   /app/src/main/jniLibs/armeabi/xxx.so

若以上文件夹没有，则自行创建。
